Comprendre les Media Queries en CSS

Qu'est-ce qu'une Media Query ?

Une media query est une fonctionnalité du CSS3 qui permet d’adapter le style d’un site web en fonction de la taille de l’écran. Elle est essentielle pour créer un site responsive, c’est-à-dire un site qui s’adapte automatiquement aux smartphones, aux tablettes et aux ordinateurs.

Grâce aux media queries, on peut modifier la disposition, la taille du texte, les couleurs ou encore la structure de la page selon la largeur de l’écran.

Affichage sur Smartphone 📱

Sur un smartphone, l’écran est petit. On privilégie donc une disposition verticale avec des blocs qui prennent toute la largeur de l’écran. Le texte est légèrement agrandi pour être lisible facilement.

Dans le fichier CSS, on utilise généralement : @media (max-width: 600px) pour cibler les petits écrans.

Affichage sur Tablette 📱

Une tablette possède un écran plus large qu’un smartphone. On peut commencer à afficher des éléments côte à côte, par exemple en utilisant Flexbox.

On utilise souvent : @media (min-width: 601px) and (max-width: 1024px) pour cibler les tablettes.

Affichage sur PC 💻

Sur un ordinateur, l’espace est plus important. On peut afficher plusieurs colonnes, réduire la taille des marges relatives et organiser le contenu de manière plus complexe.

On utilise généralement : @media (min-width: 1025px) pour les écrans larges.

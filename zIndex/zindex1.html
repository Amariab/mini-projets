Comprendre le z-index en CSS

1. Qu'est-ce que le z-index ?

Le z-index est une propriété CSS qui permet de gérer l'ordre d'empilement des éléments qui se superposent.

Plus la valeur du z-index est grande, plus l'élément sera affiché au-dessus des autres.

2. Condition importante ⚠

Le z-index fonctionne uniquement sur des éléments positionnés avec :

  • position: relative;
  • position: absolute;
  • position: fixed;
  • position: sticky;

3. Exemple visuel

Voici trois blocs superposés :

z-index: 1
z-index: 2
z-index: 3

4. Code utilisé

.box1 { z-index: 1; }
.box2 { z-index: 2; }
.box3 { z-index: 3; }

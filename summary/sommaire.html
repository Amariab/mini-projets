Mini Projet Summary / Details

1. Fonctionnement natif HTML

Ce bloc s’ouvre et se ferme automatiquement sans JavaScript. Il utilise uniquement les balises <details> et <summary>.

Il est accessible et fonctionne dans tous les navigateurs modernes.

Vous pouvez cliquer sur le titre pour l’ouvrir ou le fermer.

Le triangle à gauche est géré par le navigateur.

2. Personnalisation CSS

Le style du triangle, les bordures, les ombres, et les animations sont contrôlés avec du CSS uniquement.

3. Interaction JavaScript

Ce panneau est contrôlé dynamiquement. Un message apparaît dans la console lors de l’ouverture ou fermeture.
